# 🚀 Texas Web Development | Premium High-Conversion Landing Page

This repository contains a high-end, performance-optimized landing page built for Texas Web Development. Designed with a "conversion-first" mentality, it focuses on reducing friction and driving user action.

---

## ✨ Key Features

- **Premium:** Large, bold typography with a clear, high-impact hook.  
- **Performance Optimized:** Lightning-fast load times deployed via Vercel.  
- **SEO Ready:** Includes dynamic `sitemap.xml`, `robots.txt`, and optimized metadata.  
- **PWA Integrated:** Fully configured `manifest.json` for a native-app feel on mobile.  
- **Social Preview:** Custom Open Graph image (`og-image.jpg`) for professional sharing on LinkedIn/Twitter.

---

## 🛠️ Tech Stack

- **Markup & Styling:** HTML5 / Modern CSS — clean, semantic code with generous white space.  
- **Deployment:** Vercel for global edge delivery.
